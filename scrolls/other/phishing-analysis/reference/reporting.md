# Abuse Reporting

## Report To

| Entity | What to Report | How |
|--------|---------------|-----|
| **Registrar** | Domain abuse | WHOIS abuse contact or registrar's abuse form |
| **Hosting provider** | Hosted malicious content | Abuse contact from IP WHOIS |
| **DNS provider** | DNS services for malicious domain | Provider's abuse process |
| **Google Safe Browsing** | Phishing/malware sites | safebrowsing.google.com report form |
| **Anti-Phishing Working Group** | Phishing URLs | reportphishing@apwg.org |
| **Brand owner** | Brand impersonation | Brand's security/legal team |
| **CERT/CSIRT** | Significant threats | National/organizational CERT |

## Information to Include

- Domain name and URL(s)
- Type of abuse (phishing, malware, spam, impersonation)
- Screenshots of malicious content
- WHOIS/RDAP data
- DNS records showing infrastructure
- Timeline (when discovered, estimated start date)
- Any victim information (redacted)
