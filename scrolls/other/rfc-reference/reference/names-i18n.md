# Domain Names & Internationalization RFCs

| RFC | Title | Status | Summary |
|-----|-------|--------|---------|
| **1123** | Requirements for Internet Hosts | Standard | Relaxed hostname rules: labels may start with a digit. Defines the LDH (Letters-Digits-Hyphens) rule. |
| **5890** | Internationalized Domain Names for Applications (IDNA2008) | Standard | Framework for internationalized domain names. Defines valid Unicode label categories. |
| **5891** | IDNA2008 Protocol | Standard | Registration and lookup procedures for IDN. Converting between U-labels (Unicode) and A-labels (Punycode). |
| **5892** | The Unicode Code Points and IDNA2008 | Standard | Rules for which Unicode code points are valid in domain labels. |
| **5893** | Right-to-Left Scripts for IDNA2008 | Standard | Bidi rules for domain labels containing right-to-left characters (Arabic, Hebrew). |
| **3492** | Punycode | Standard | The Punycode encoding algorithm: converts Unicode labels to ASCII-compatible encoding prefixed with `xn--`. |
