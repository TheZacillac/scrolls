# RFC Reference — Domain-Relevant Standards Index

## What is the RFC Reference?

A curated index of RFCs relevant to domain names, DNS, email authentication, and related protocols. Each entry includes the RFC number, title, status, and a plain-language summary for AI agents to reference when answering technical questions.

## Reference

- [DNS Core](reference/dns-core.md) — Foundational DNS RFCs (1034, 1035, 2181, 6891, etc.)
- [DNSSEC](reference/dnssec.md) — DNS Security Extensions (4033–4035, 5155, 8624, etc.)
- [Email Authentication](reference/email-auth.md) — SPF, DKIM, DMARC, ARC (7208, 6376, 7489, etc.)
- [WHOIS & RDAP](reference/whois-rdap.md) — Registration data protocols (3912, 7480–7484, 9082–9083)
- [Domain Names & Internationalization](reference/names-i18n.md) — IDN, Punycode, IDNA2008 (5890–5893, 3492)
- [DNS Operations](reference/operations.md) — Transport, best practices, and record types (7858, 8484, 6698, etc.)
- [Record Types Quick Reference](reference/record-types.md) — All DNS record types with RFC source and purpose
