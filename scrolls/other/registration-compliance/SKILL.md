# Registration Compliance — ICANN Policies & Domain Lifecycle

## What is Registration Compliance?

This skill covers the regulatory framework governing domain name registration: ICANN policies, registrar obligations, dispute resolution procedures, transfer rules, and the domain lifecycle. Essential reference for understanding why domains behave certain ways and what recourse is available.

## Quick Start

```bash
seer lookup example.com   # Check status codes, registrar, expiry
seer whois example.com    # Full WHOIS details
seer rdap example.com     # Structured RDAP data
```
